# Game Concept

## Genre
**Roguelike Shoot 'em Up** (Bullet Heaven / Survivor-like).

## Core Gameplay Loop
1.  **Explore**: Navigate procedurally generated dungeons using the 32x32 tile grid.
2.  **Fight**: Battle waves of enemies (6 types) using directional shooting.
3.  **Loot**: Collect XP (crystals), weapons, and power-ups.
4.  **Upgrade**: Use UI panels to upgrade stats and equipment.

## Mechanics
- **Movement**: Top-down, 8-directional (or 4-directional) movement.
- **Combat**: Projectile-based shooting. Weapons overlay on the character.
- **Progression**: XP collection leads to level-ups and new abilities.
