## Technical Specifications

### File Format
- **PNG** with transparency
- 32-bit color depth

### Size Guidelines
- Icons: 32x32 pixels
- Portraits: 64x64 pixels  
- Sprites: 32x32 or 48x48 pixels
- Backgrounds: 800x600 pixels
- UI panels: Variable, power of 2 preferred

### Naming Convention
```
icon_upgrade_<name>.png       (e.g., icon_upgrade_damage.png)
icon_weapon_<name>.png        (e.g., icon_weapon_axe.png)
icon_achievement_<name>.png   (e.g., icon_achievement_first_blood.png)
portrait_<character>.png      (e.g., portrait_knight.png)
bg_<scene>.png                (e.g., bg_main_menu.png)
deco_<type>_<variant>.png     (e.g., deco_skull_01.png)
```

## AI Image Generation Prompts

Here are example prompts for generating assets:

### Icon Style Prompt Template
```
Pixel art icon, 32x32, [SUBJECT], dungeon/roguelike style, 
dark background with glow, fantasy RPG aesthetic, 
clean lines, limited color palette, no anti-aliasing
```

### Character Portrait Template
```
Pixel art portrait, 64x64, [CHARACTER] face, 
roguelike fantasy game, dark dungeon atmosphere,
detailed but readable at small size, 
consistent with retro RPG style
```

### Decoration Template
```
Pixel art game asset, 32x32, [OBJECT], 
top-down dungeon view, dark stone floor,
roguelike/dungeon crawler style,
tileable edges if applicable
```
